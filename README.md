# 🎵 Miku MIDI Phrase Pack Generator

A browser-based MIDI phrase generator for vocal melodies inspired by Miku-style songwriting.

No installation required. Just open the HTML file in your browser and start generating MIDI phrases.

## Features

- Browser-based (HTML + JavaScript)
- Generate 2 / 4 / 8 bar vocal phrases
- Phrase Types
  - Verse
  - Pre-Chorus
  - Chorus
  - Bridge
  - Hook
  - Ending
- Vocal Styles
  - Cute
  - Soft
  - Power
  - Dark
  - Emotional
  - Electronic
  - Pop
  - Rock
- Random or fixed chord progression
- Rhythm weight control
- Seed-based reproducible generation
- Piano Roll preview
- WebAudio playback
- Standard MIDI File (Format 0)
- Batch generation
- ZIP export

## Usage

1. Open `miku_midi_phrase_pack_generator.html`
2. Select phrase settings.
3. Click **Generate**.
4. Preview the melody.
5. Download MIDI or ZIP.

## Specifications

- MIDI Format 0
- 120 PPQ
- Tempo 120 BPM
- Major Scale
- Browser only
